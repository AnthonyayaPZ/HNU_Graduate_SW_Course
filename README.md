## 前言
研一上学期的最后一周，本人一周内六门考核，一门汇报 + 两门闭卷考 + 三门开卷考，复习得欲仙欲死，所以决定把自己的复习经验、资料，以及自己对于课程的评价分享一下，希望能给软件方向的学弟学妹们的选课提供一下参考，复习提供一些方向。

## 1. 敏捷软件开发：

- 课程代码：B2510155M
- 授课教师：金敏
- 课程考核：课堂表现*50% + 小组合作*20% + 期末考试(开卷)*30%

个人评价：老师课上的很好，但是这个小组合作很折磨，具体形式是针对授课内容提问题或者给实践项目，要求小组合作课上20分钟内完成，整个学期差不多有5、6次，所以不太好混。
复习建议：首先是开卷，然后大题全是开放题，所以考前不用花太多时间复习，在雨课堂把课堂测试那几个选择题看看就行了。大题是一个案例分析 + 一个AIGC经验分享。

## 3. 微服务与云计算：

- 课程代码：B2510022M
- 授课教师：杨金民
- 课程考核：课堂表现(考勤+汇报)*25% + 作业*15% + 期末考试(闭卷)*60%

个人评价：老师说话不是很清楚，而且经常是讲PPT讲到一半就开始即兴发挥侃山侃地，但是非常建议各位去看一下杨老师主页上的那几篇文章，https://csee.hnu.edu.cn/people/yangjinmin，除开讲课质量不谈还是一个很好的老师。
复习建议：考勤每节课前点名，点完名可润；汇报很恐怖，老师会追着你问问题，但是是可选的；考试说是闭卷，但题目就是课上给的思考题，提前背就好。

## 4. IT项目管理：

- 课程代码：B2510011M
- 授课教师：胡军
- 课程考核：课堂表现(提问)*20% + 雨课堂答题*10% + 期末考试(开卷)*70%

个人评价：选课前听说很水，实际上每节课会有雨课堂测试，还会随机点名。
复习建议：最宽松的开卷，允许带电子设备，说是不给联网但你要联也管不了你，把那几个大题提前过一下就好：PERT、关键路径、决策树、挣值分析、历时估算

## 5. 研究生英语(托福)：

- 课程代码：20004M
- 授课教师：杨艳（8388）
- 课程考核：平时成绩*40% + 期末考试*60%(口试*30% + 笔试*30%)

个人评价：上过事最多的课，一大堆课堂作业，虽然老师人还不错，但我还是要说：跑吧，不推荐
复习建议：没啥好说的，把她布置的那一堆课堂作业都完成的话，成绩绝对不会差。但是还是说一下考试形式：期末口试是看别人雅思口语考试的视频，然后估计视频中的考生能拿几分，猜的越接近分越高；期末笔试就是托福听力题，但是可以提前看选项。

## 6. 并行算法分析与设计：

- 课程代码：B2510005M
- 授课教师：吴帆
- 课程考核：期末考试(开卷) + 汇报，占比不清楚

个人评价：老师上课上的还挺好的，几个算法都讲的很清楚，时不时会点个名，
复习建议：开卷，考的题目全是PPT上的，提前把那几个算法过一遍就好：前缀和、归并排序、矩阵乘法，真要提前熟悉这几个算法具体在PPT的哪一页，不然到时候抄都抄不过来，题量大的一
